TeamOne sidebar #

You can access TeamOne sidebar from the Extensions menu:

The sidebar toolbar allows you to select the current environment, refresh the config from TEAMONE sheet (if you made any changes) and do several quick actions from the main menu:

  • Search TM1 model
  • Check server connection
  • Update security, if you defined a server secruity sheet
  • Save data on TM1 server

SSL certificate #

To prevent SSL warnings in your browser we recommend configuring a custom SSL certificate (signed by a trusted certificate authority) for your TM1 server. Here are the IBM instructions for How to Configure SSL on IBM Planning Analytics

TEAMONE config sheet #

TeamOne add-on stores all the settings on a sheet called TEAMONE. You can add your TM1 connections and tasks as static values or using Sheets formulas. The parameters are entered in column A and the values are in column B. You can also use # for comments or to just to skip some lines. Configuration menu allows to quickly add new TM1 connections and tasks

Adding TM1 connections #

To connecte to TM1 you need to add a new tm1env definition. You can do it manually or by using TeamOne > Configuration > Add new TM1 connection menu. Then edit the default parameters:

  • change TeamOneTest to the desired connection name
  • protocol use https if useSSL=T in tm1s.cfg file, otherwise use http
  • host specify your TM1 server host FQDN
  • port should be equal to HTTPPortNumber in tm1s.cfg file
  • if using IntegratedSecurityMode 4 or 5 specify namespace configured in Cognos Analytics (usually "LDAP")
  • secret is a base64 encoded username and password, the default one is for admin user
