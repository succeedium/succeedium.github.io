Current version #

Version 1.0 was release on May 4, 2021

We are happy to announce TeamOne v1.0, which includes the next features:

  • Multiple TM1 environments, add as many TM1 environments as you need, you can quickly switch to the one you want to work with
  • Tasks, simply add TM1 processes and parameter values which can be calculated using Google Sheets formulas
  • TM1 server dashboard, see what is going on with your TM1 server in real time
  • TM1 log, see and search TM1 server message log as you type
  • Users management, add, delete users, modify permissions, see who is in the system right now
  • Deployments, quickly compare two environments, see the code difference and promote the processes from one environment to another one
  • Code search, search for any keyword through all the processes as you type
  • TM1 Console, type and execute your TI codem syntax highlighing is included
