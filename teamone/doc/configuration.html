Getting started #

SSL configuration for your TM1 server #

To prevent SSL warnings in your browser we recommend configuring custom SSL certificate for your TM1 server

TEAMONE configuration sheet #

TeamOne add-on stores all the settings on a sheet called TEAMONE. You can add your TM1 connections, tasks and views as static values or using Sheets formulas. The parameters are entered in column A and the values are in column B. You can also use # for comments.

Adding TM1 connections #

Add your first TM1 connection by using TeamOne > Configuration > Add new TM1 connection menu:

Then edit the default connection parameters:

  • change TeamOneTest to the desired connection name
  • protocol use https if useSSL=T in tm1s.cfg file, otherwise use http
  • host specify your TM1 server host FQDN
  • port should be equal to HTTPPortNumber in tm1s.cfg file
  • if using IntegratedSecurityMode 4 or 5 specify namespace configured in Cognos Analytics (usually "LDAP")
  • secret is a base64 encoded username and password, the default one is for admin user
