# Tasks

Tasks allow you to create automations for TM1 TI processes

# Define a new task

The simplest way is to start with a template using TeamOne > Configuration > Add new TM1 task menu

# Making tasks dynamic

You can use Sheets formula to calculate task parameter names and values. A common example would be to use =YEAR(TODAY()) formula to get the current calendar year
