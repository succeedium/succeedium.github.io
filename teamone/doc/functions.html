Functions #

Functions tab extends sheets with TM1 specific functionality. All the functions will be performed on the current/active sheet

TEAMONE recalculate #

Recalculates all TEAMONE formulas on the current sheet. TEAMONE formula allows to bring data from TM1 server. You should have at least one TM1 connection configured. TEAMONE formula works in 2 steps

STEP 1. Define a formula #

TEAMONE formula expects the next arguments:

  • cube connection - "TM1 connection:cube name" string, where TM1 connection is one of the configurated connections. You can also specify only the cube name to fetch data from the currently selected TM1 connection in TeamOne sidebar.
  • rangeRows - rows range, select the cells represeting your rows
  • rangeColumns - columns range, select the cells represeting your columns
  • rangeTitles - rows range, select the cells represeting your context elements For rows and columns you can select multiple cells, a single cell or even no cells, use an empty string or leave the argument empty:
// No rows and no titles example (arguments #2 and #4 are skipped):
=TEAMONE('Revenue', , C1:F1)

Range

A range should represent TM1 dimension element names or aliases. You can use 2 dimensional ranges for rows and columns (to use multiple cube dimensions), but titles range should only include either a single cell or a one dimensional range. You should specify at least 1 row or 1 column or 1 title range

Dimensions and elements

You do not need to include elements for all the cube dimensions. STEP 2. Recalculate the values #

Go to TeamOne sidebar > Functions tab and click TEAMONE recalculate. Depending on how many formulas you have on the current sheet and your TM1 model and data complexity, it may take a few seconds or much longer to fetch all the requested data.

Increase indent #

Being a really great tool, sheets are missing 2 simple functions to increase and decrease cell indentation. Increase indent adds 2 spaces (on the left) for each cell value in the selected range.

Decrease indent #

Opposite to the function above, Decrease indent removes 2 spaces (from the left) for each cell value in the selected range.
