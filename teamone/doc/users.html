Users #

You can see all users, create new users, enable\disable users, see who is active, logout, delete and manage permissions on the users tab of TeamOne sidebar. Click any user to modify the permissions, delete, enable\disable or logout him.

Managing users and groups #

Modifying user permissions #

Use checkboxes to specify user's groups. For your convenience, the selected groups are displayed first, so if you select a new group it will move to the top. To change this behaviour click the Group header to order groups by name

Deleting users #

Use the Delete button at the bottom to delete the selected user

Disabling users #

You can disable all non-admin users from the users menu or you can disable individual users by clicking the Disable user button at the bottom of the edit user screen. The button does not show up if a user is disabled. The button is gery out if a user belongs to ADMIN security group.

Enabling users #

You can enable all disabled users from the users menu or you can enable individual users by clicking the Enable user button at the bottom of the edit user screen. The button does not show up if a user is enabled.

Logging out users #

Use the Logout button at the bottom to logout the selected user. The button is displayed only for active users.
