Users #

Tasks allow you to create automations for different TM1 routines (running TI processes, import and export TM1 data, etc.). Using TeamOne > Configuration menu can quickly add new tasks templates to TEAMONE configuration sheet and then modify them as needed. You can use Sheets formulas to make tasks dynamic. A common example would be to use =YEAR(TODAY()) formula to get the current calendar year and use it as a parameter for TM1 process task. All the defined tasks can be run from the TeamOne sidebar:

Each task is defined by specifying a ::tasktype in column A and task name in column B. To execute a task you simply click it in TeamOne sidebar.
