# What is TeamOne?

Sheets + IBM Planning Analytics™ = TeamOne. TeamOne is an add-on that allows you to work with IBM Planning Analytics™ directly from Google Sheets™.

IBM Planning Analytics is a software product developed by IBM and Google Sheets is a registered trademark owned by Google LLC.

Excel on steroids - old famous description of TM1, used many years ago. Since then IBM Planning Analytics™ took it to another level. Similarly Google Workspace™ (formerly G Suite™) has been building products to help people transform the way they work. TeamOne brings IBM Planning Analytics™ into Google Sheets™. See below what it can do.

# Planning Analytics data

Viewing TM1 data directly in Sheets was the main reason to start TeamOne add-on. Similar to TM1 Perspectives for Excel, TeamOne will allow to view and update TM1 data, but directly in Google Sheets™. Though this functionality is still under development, in the current version you can already get TM1 data into a sheet by specifying an MDX expression.

In the next version you will be able to use TEAMONE formula to fetch TM1 data slice by specifying a cube name and dimension elements.

# Multi-environment support

An average TM1 setup usually includes at least 3 environments: DEV, QA, PROD. Some companies use even more (Sandbox, Test, UAT, PrePROD, etc.). Keeping TM1 models organized and in sync is not a trivial task and requires using some 3rd party or custom developed tools. In TeamOne you can add as many connections as you need and easily manage or deploy items by selecting your current TM1 source and target instances.

# Tasks

Open a new period, reload data, recalculate cube, copy scenario, refresh feeders... Sounds familiar? Now all those routines can be automated using TeamOne tasks. Simply define a new task, specify a process name and parameters and now you can run it in any environment. Moreover you can use Goole Sheets formulas to make it fully dynamic.

# System dashboard

How would you like to have tm1top right in Google Sheets? Now you got it. See active sessions and execution time just on right sidebar

# Users & Groups

As you would expect, you can see all TM1 users and their groups in TeamOne side bar too. But what about:

  • see who is in the system now
  • managing groups assignments
  • adding new users
  • deleting users
  • or syncing users & groups from a sheet

TIP

Create a new sheet with users and their security groups. Use Protected sheets and ranges to grant permissions for business managers who can modify the assignments. Now you are one click away to sync the permissions for the entered users. Need to manage permissions for a different environment or TM1 model? Create a new sheet...
