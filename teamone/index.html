What is TeamOne? #

Sheets + IBM Planning Analytics™ = TeamOne. TeamOne is an add-on that allows you to work with IBM Planning Analytics™ directly from Google Sheets™.

IBM Planning Analytics is a software product developed by IBM and Google Sheets is a registered trademark owned by Google LLC.

Excel on steroids - old famous description of TM1, used many years ago. Since then IBM Planning Analytics™ took it to another level. Similarly Google Workspace™ (formerly G Suite™) has been building products to help people transform the way they work. TeamOne brings IBM Planning Analytics™ into Google Sheets™. See below what it can do.

Planning Analytics data #

As you would expect, the main add-on feature is to view TM1 data directly in Sheets. You can do it using =TEAMONE formula or tm1view task.

Multi-environment support #

An average TM1 setup usually includes at least 3 environments: DEV, QA, PROD. Some companies use even more (Sandbox, Test, UAT, PrePROD, etc.). Keeping TM1 models organized and in sync is not a trivial task and requires using some 3rd party or custom developed tools. In TeamOne you can add as many connections as you need and easily manage or deploy items by selecting your current TM1 source and target instances.

Tasks #

Open a new period, reload data, recalculate cube, copy scenario, refresh feeders... Sounds familiar? Now all those routines can be automated using TeamOne tasks, moreover, it can be done by business users. Simply define a new task, specify a process name and parameters and now you can run it in any environment. Moreover you can use Goole Sheets formulas to make it fully dynamic.

System dashboard #

How would you like to have tm1top right in Google Sheets? Now you got it. See active sessions and execution time just on right sidebar

Users & Groups #

As you would expect, you can see all TM1 users and their groups in TeamOne side bar too. But what about:

  • see who is in the system now
  • managing groups assignments
  • adding new users
  • deleting users
  • or syncing users & groups from a sheet

TIP

Create a new sheet with users & security groups. Use Protected sheets and ranges to grant permissions for business managers who can modify the assignments. Now you are one click away to sync the permissions for the entered users. Need to manage permissions for a different environment or TM1 model? Create a new sheet with users...

TM1 Message log #

Something failed again or you just wanted to confirm something in the server log? It is at the same place, your TeamOne side bar can display and filter the message log as you type

Documentation #

The project is almost done, but the documentation has not been even started? Or it just never gets updated? No suprise, this is common for all many TM1 projects. What about auto-documentation? If you follow TeamOne Developement and Naming Conventnion guide, not only will your TM1 code be much cleaner and easier to read, but also it will be auto-documented as well in real-time.
