FAQ ​

What access does TeamOne require? ​

When you install TeamOne add-on it will ask you for the next permissions:

PermissionDescription
See your primary Google Account email addressTeamOne needs your email address to verify the license status. This is the only personal information TeamOne has access to.
Display and run third-party web content in prompts and sidebars inside Google applicationsTeamOne requires permission to display a sidebar and additional screens (within Google Sheets application) to enable its functionality.
Connect to an external serviceTeamOne needs permission to connect to IBM Planning Analytics REST API in order to integrate IBM Planning Analytics data into Google Sheets.
View and manage spreadsheets that this application has been installed inTeamOne needs permission to modify the current Google Sheets document, such as adding new tabs and loading data from IBM Planning Analytics. TeamOne cannot access your other documents.

TeamOne’s use and transfer of information received from Google APIs to any other app will adhere to Google API Services User Data Policy, including the Limited Use requirements.

Do I need to buy any Google subscription for TeamOne? ​

No. Even a free gmail account would work for individual licenses. Though if you purchase a Google Workspace plan, you will be able to install and configure TeamOne for all your users.

Does any server side software need to be installed? ​

No. TeamOne lives in Google Cloud and does not require anything to be installed on IBM PA (TM1) server or you computer.

Will any of my data be sent or stored on any 3rd party servers? ​

TeamOne connects directly from your browser to IBM Planning Analytics / TM1 server. No data is being sent or stored on any other servers.

Can TeamOne connect to IBM Planning Analytics Cloud ​

Yes, TeamOne can connect to IBM PA cloud. You can configure a Single-Sign-On or use a non-interactive user account.

What IBM Planning Analytics version is supported ​

Even though some functionality may work in the older versions, TeamOne add-on officially supports IBM Planning Analytics version 2.0.9 or newer.

Which TM1 IntegratedSecurityMode does TeamOne support? ​

TeamOne supports the next TM1 integrated security modes: 1 (native TM1 security), 4 and 5 (Cognos security)

Is security tied to IBM Planning Analytics? ​

Yes, when connecting from Google Sheets, users can only access data and perform actions based on their roles and permissions configured in IBM Planning Analytics.

Do users need technical skills to query IBM Planning Analytics data? ​

No, TeamOne allows users to focus on business insights without requiring technical skills.

  • TEAMONE() Function: Fetches data without needing dimension names, element orders, or hierarchy details. TeamOne automatically identifies relevant dimensions, applies default hierarchy members, and alerts users to errors like invalid element names or mismatched dimensions.
  • MDX() Function & Explorations: Enable users to build dynamic queries using simplified MDX expressions (TeamOne MDX Mods) without technical expertise.

With TeamOne, users can analyze data and make informed decisions without worrying about complex query structures.

Learn more: MDX Mods

Does TeamOne provide a formula builder? ​

Yes, TeamOne simplifies formula creation and exploration setup with intuitive tools.

  • Formula & Exploration Generation: Automatically generates formulas and explorations based on existing cube views. Users can also use an interactive interface to select dimension elements, design query layouts, and build reports effortlessly—no technical expertise required.
  • Automated Dimension Detection: The TEAMONE() function intelligently identifies relevant dimensions, eliminating manual input and streamlining data retrieval and analysis.

Is training available for TeamOne users? ​

Yes, we offer training to help users effectively navigate and utilize TeamOne.

Where is TeamOne hosted? What about data residency, compliance, and latency? ​

TeamOne runs on Google Cloud Platform (GCP) like all Google Workspace add-ons. However, data in Google Sheets follows your Google Workspace settings, ensuring compliance with your organization's data residency requirements.

How does TeamOne protect data? Does it use encryption and meet security standards? ​

TeamOne secures data in transit with TLS 1.2+ and encrypts data at rest using AES-256. It adheres to SOC 2 standards and is undergoing a continuous 12-month SOC 2 Type 2 audit.

Does TeamOne integrate with other Google Workspace applications? ​

TeamOne doesn’t directly integrate with other Google Workspace apps but leverages Google Sheets' native connections to Docs, Slides, and Forms. Linked data updates automatically, allowing seamless synchronization across applications.

How is TeamOne deployed and configured? ​

TeamOne is a Google Workspace add-on that can be installed from the Google Workspace Marketplace with no technical expertise required. Administrators can pre-install it for users via the Google Admin console and configure IBM Planning Analytics connections in advance, enabling immediate access.

What are the ongoing maintenance and upgrade requirements for TeamOne? ​

TeamOne receives automatic updates via the Google Workspace Marketplace, ensuring access to the latest features, security patches, and performance improvements. Updates are applied seamlessly in the background, requiring no downtime or manual intervention.

Does TeamOne affect Google Sheets' performance with large datasets or complex formulas? ​

No, TeamOne is designed for efficiency. It recalculates only when inputs change and uses data chunking to handle large datasets smoothly without slowing down Google Sheets.

Does TeamOne require a constant internet connection? ​

Yes, TeamOne needs an active internet connection to function, as it communicates with the Google Sheets API.

What are the top benefits of TeamOne? ​

  • User-Friendly Formulas: Intuitive formulas eliminate the need for technical knowledge, automatically handling dimension selection and cube structures.
  • Self-Service Reporting: Users can access IBM Planning Analytics (IPA) data and create reports in Google Sheets without relying on IT, enabling agility and faster decision-making.
  • Seamless Collaboration: Works within Google Sheets, allowing easy sharing and collaboration without additional software installation—accessible on any device.
