Model search ​

Searching ​

Use the search bar to search through every process/cube in a selected server. You can use the dropdowns at the top left to select which server and asset type you want to search through. You will be shown a list of asset that include the searched keyword.

Searching through an asset ​

To search through an asset, you can go through the tabs at the top, a tab with a green circle signifies that your search term is included in that tab. You search term will be highlighted. To navigate through every instance of your search term in an asset, you can click on the "next" button.

Keyboard shortcuts ​

Here's a list of helpful keyboard shortcuts to navigate your search:

  • Ctrl-F / Cmd-F: Search
  • Ctrl-G / Cmd-G: Find next
  • Shift-Ctrl-G / Shift-Cmd-G: Find previous

Refreshing cache ​

For faster search results, the search data is stored on our servers. To refresh the cache on our servers, an admin user can click on the "Refresh cache" button.

System objects ​

You can disable/enable searching through system objects by toggling the "System objects" switch

Search and replace ​

The search and replace functionality allows you to search for a keyword in the selected server and replace every instance of that keyword in the selected asset type with another keyword. To open the search and replace and menu, click on the "Search and replace button".

Enter your search keyword and your replace keyword, and click on replace all.
