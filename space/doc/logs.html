Server logs ​

Message log ​

Message logs can be filtered by level through the level dropdown, by start and end date, and by messages containing a certain string. Each page only displays 1000 logs, you can change page using the arrows at the bottom right of the screen.

Transaction log ​

Transaction logs can be filtered by Cube and User through their respective dropdowns, aswell as by a date range and elements.

Audit log ​

Similar to the other log types, audit logs can be filtered by a date range. It can also be filtered by the log description, user, and object type.

Updating logs ​

Logs can be updated in the SPACE settings, however this can only be done by users with admin tokens. To update logs, go to the "Content" tab of the SPACE settings, select the log you want to update, and click on "Update".
