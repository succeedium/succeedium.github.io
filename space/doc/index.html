Getting started ​

What is SPACE? ​

The Succeedium Planning Analytics Cloud Extension (SPACE) is a comprehensive set of tools designed to help companies maximize the potential of IBM Planning Analytics. SPACE is integrated into IBM Planning Analytics Workspace and offers several functionalities such as:

  • Python and Jupyter Notebook Integration:
    Developers can create, edit, and execute Python scripts and Jupyter Notebooks directly within PAW, without the need for additional installations on servers or local machines. This integration facilitates advanced data analysis, automation, and complex calculations.

  • Automated Model Documentation:
    SPACE generates detailed, interactive documentation of TM1 models, allowing users to customize and update documentation dynamically, thereby enhancing transparency and understanding of complex TM1 structures.

  • TM1 Lifecycle Manager:
    This feature enables the comparison of TM1 models and the deployment of changes directly or through organized releases. It supports the promotion of various TM1 objects, including processes, chores, cubes, dimensions, views, subsets, attributes, and data, streamlining model development, testing, and deployment.

  • Advanced Git Integration:
    Tailored for TM1 projects, SPACE allows for seamless connection to major Git platforms, facilitating collaborative development workflows and robust version control.

  • Enhanced Log Management:
    SPACE provides rapid search capabilities for TM1 Server, Transaction, and Audit logs. Users, including non-admins, can save searches, generate reports, and quickly access important log information, improving transparency and auditability.

  • User Access Troubleshooting:
    SPACE can automatically scan all cubes, dimensions, hierarchies, elements, and data security settings to explain user access levels, simplifying the troubleshooting process.

SPACE architecture ​

SPACE is a containerized application designed for seamless deployment across a variety of environments. It can be installed on leading cloud platforms such as AWS, Google Cloud Platform (GCP), Microsoft Azure, and IBM Cloud, or hosted on-premises for organizations using IBM Planning Analytics Local. With its ability to connect to multiple IBM Planning Analytics (TM1) servers, SPACE empowers users to seamlessly manage and interact with multiple environments from a unified interface, streamlining workflows and enhancing productivity.
