Succeedium TeamOne is a Google Sheets add-on for IBM Planning Analytics. Similar to TM1 Perspectives it allows to read and write data from your TM1 cubes, but directly in Google Sheets. It also contains a small sidebar which gives you a full control on your TM1 servers, including: user sessions dashboard (similar to tm1top, but with a nice formatted view), security management (users and groups assignments), TI process execution, deployment (from model A to model B without TM1 server restart), server health status (CPU, RAM and disk usage), server message log, file uploads to TM1 server, TI code version control (with ability to compare the changes), automated model documentation, performance testing.

Succeedium TM1 Framework is a set of TM1 TurboIntegrator processes and Applications to spead up your TM1 development and automate your TM1 model administration

Succeedium TM1 Excel Tools is an Excel add-on for IBM Planning Analytics that allows to manage your TM1 Perspectives reports including a global search and replace and deployments. You can also search your TM1 code (TurboIntegrator processes and cube calculation rules) as well as deploy TM1 objects from model A to model B directly from Excel without TM1 server restart.
