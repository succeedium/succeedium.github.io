About

TeamOne is a Google Sheets add-on for IBM Planning Analytics. Similar to TM1 Perspectives it allows to read and write data from your TM1 cubes, but directly in Google Sheets. It also contains a small sidebar which gives you a full control on your TM1 servers, including:

  • Session dashboard (similar to tm1top, but with a nice formatted view)
  • Security management (users and groups assignments)
  • TI process execution
  • Deployment (from model A to model B without TM1 server restart)

If you install TeamOne agent somewhere in your network, this feature list will go on:

  • Server health status (CPU, RAM and disk usage)
  • File uploads to TM1 server
  • TI code version control (with ability to compare the changes)
  • Automated model documentation
  • TM1 performance testing

Privacy policy

TeamOne Google Sheets add-on does not store any data or personal information, all the data is retrieved in real time directly from your TM1 server REST API or through TeamOne agent and is either displayed in a sidebar or loaded on a Google Sheet tab
